# 📜 書香閱讀 (MyRead) 變更日誌 (CHANGELOG)

### [V3.0] - 2026-03-17 (當前版本)
#### 🚀 新增功能
- **全欄位編輯**: 書籍詳情頁支援即時修改所有資訊。
- **智慧序號 (BKxx)**: 修正 GAS 邏輯，自動生成支援補零的專業序號格式。
- **高清封面更新**: 支援在編輯模式中隨時更換封圖網址。
- **草圖佈局重構**: 依照使用者手繪草稿，重組首頁表單與統計展示區域。

#### 🔧 優化與修復
- **引擎升級**: 全面對接 **Gemini 2.5 Flash** 模型，解決 404 報錯問題。
- **視覺放大**: 針對導航、按鈕與內容文字進行全站字級提升 (放大 2 級)。
- **資料讀取**: 修正 GAS API 讀取舊資料分頁的邏輯。

#### 🚀 新增功能
- **AI 深度摘要**: 整合 Google Apps Script 呼叫 Gemini 進行 200 字以上摘要。

- **博客來爬蟲**: 自動抓取網路書源與簡介。
