# 🎨 書香閱讀 (MyRead) 介面與設計規範 (UI_UX_GUIDE)

## 1. 核心風格：Light Glassmorphism (亮色玻璃擬態)
致力於營造一種「明亮書房」的清爽感。利用全白半透明層次、更強的模糊效果與細膩的櫻花粉邊框，展現現代、乾淨且高級的數位閱讀空間。

### 🎨 配色方案 (Color Palette)
| 用途 | 顏色代碼 | 說明 |
| :--- | :--- | :--- |
| **背景主色** | `#fbfbfd` | 極簡米白背景 |
| **卡片背景** | `rgba(255, 255, 255, 0.7)` | 白色半透明磨砂卡片 |
| **品牌強調色** | `#fb6f92` | 櫻花粉 (Core Pink) |
| **懸停發光** | `rgba(251, 111, 146, 0.2)` | 柔和粉色光暈 |
| **文字主要** | `#1d1d1f` | 深炭灰色 (High Readability) |
| **文字次要** | `#636366` | 中灰色 |

### 🔡 字體規範 (Typography)
- **英文/數字**: `Outfit`, `Inter` (強調現代感與幾何美學)。
- **中文內容**: `Noto Sans TC`, `system-ui` (確保跨平台閱讀流暢)。
- **標誌/標題**: 800 (Extra Bold)。
- **內文正文**: 400 (Regular)，行距定為 `1.9` 以利長文摘要閱讀。

## 2. 組件行為 (Component Logic)
- **卡片設計**: 統一 `28px` 圓角，邊框使用 `1px solid rgba(251, 111, 146, 0.15)` 增加品牌延伸感。
- **編輯模式**: 點擊「編輯資料」後，UI 呈現聚焦狀態，輸入框變為實色背景，給予明確的互動反饋。
- **陰影系統**: 捨棄深色重陰影，改用 `rgba(0,0,0,0.06)` 的平滑長陰影。

## 3. RWD 響應式邏輯
- **Desktop (900px+)**: 表單與統計圖採併排顯示。
- **Tablet/Mobile**: 
  - 自動切換為單欄垂直佈局。
  - 統計圖表寬度 100%。
  - 頂部導航簡化為圖示模式。

## 4. 微交互 (Micro-Animations)
- **Hover**: 所有按鈕需具備 `scale(1.02)` 與 `filter: brightness(1.1)` 效果。
- **Page Transitions**: 使用 CSS Keyframes `fadeInDown` 處理視窗切换，確保視覺不跳轉。
